# Dandeli – Forest‑Inspired Adventure Website

## Overview
A modern React single‑page application showcasing the adventure tourism destination **Dandeli** (India). The site features a deep‑forest, earthy visual design with custom Tailwind CSS variables, subtle textured backgrounds, and smooth card‑hover effects.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS v4 with custom `@theme` variables
- **Languages**: JavaScript (ES6+), HTML5
